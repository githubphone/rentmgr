# 出租屋人脸识别管理模块 - 规格说明书

## 1. 项目概述

### 1.1 项目名称
水口出租屋人脸识别管理模块

### 1.2 项目类型
基于人脸识别技术的社会治理管理系统

### 1.3 核心功能概要
通过人脸识别技术实现对出租屋进出人员的自动识别和陌生人告警管理，建立高效的告警推送和核查处置流程。

### 1.4 目标用户
- 街道办管理人员
- 网格员
- 房东/物业
- 派出所工作人员

## 2. 功能范围

### 2.1 后台管理端功能模块

#### 2.1.1 房源管理模块
- 出租屋CRUD操作
- 批量导入（Excel）
- 设备关联管理
- 按区域/状态筛选
- 地图定位展示

#### 2.1.2 住户管理模块
- 住户CRUD操作
- 人脸照片采集与上传
- 批量入住/退房登记
- 到期提醒功能
- 按出租屋/状态/类型筛选

#### 2.1.3 通行记录模块
- 实时通行记录展示
- 查询与导出功能
- 通行趋势统计
- 热力图数据展示

#### 2.1.4 告警管理模块
- 告警列表（待处理/处理中/已处理）
- 告警处理与转交
- 核查结果录入
- 告警统计

#### 2.1.5 统计分析模块
- 今日通行概览
- 陌生人高频区域TOP10
- 告警处理时效统计
- 住户登记率统计
- 设备在线率统计

#### 2.1.6 设备管理模块
- 设备列表与状态监控
- 设备信息配置
- 设备心跳检测

#### 2.1.7 地图可视化模块
- 出租屋地图标记
- 设备地图标记
- 告警实时标注
- 通行热力图
- 路径规划

### 2.2 房东小程序端功能
- 扫码绑定出租屋
- 住户自主登记（房东审核）
- AI辅助信息采集
- 智能提醒推送
- 通行动态感知
- 一键联系网格员

### 2.3 扩展模块
- 重点人员管控
- 网格员管理
- 三级联动体系

## 3. 技术架构

### 3.1 技术栈

| 层级 | 技术选型 |
|------|----------|
| 后端框架 | Spring Boot 4.0 + MyBatis 4.0 |
| 数据库 | PostgreSQL |
| 前端框架 | Vue 3 + Element Plus |
| 地图服务 | 高德地图 JavaScript API |
| 人脸识别 | 对接政府人脸比对平台API |
| 消息推送 | WebSocket + 极光推送 |
| 房东端 | 微信小程序 |
| 项目基础 | RuoYi 3.9.1 |

### 3.2 数据库表

#### 核心业务表
- `rental_houses` - 出租屋表
- `rental_residents` - 住户表
- `rental_face_photos` - 人脸照片表
- `rental_access_logs` - 通行记录表
- `rental_alerts` - 告警记录表
- `rental_verifications` - 核查记录表
- `rental_devices` - 设备表

#### 扩展表
- `rental_key_persons` - 重点人员信息表
- `rental_person_watched` - 布控记录表
- `rental_person_alerts` - 重点人员告警表
- `rental_person_tracks` - 轨迹记录表

### 3.3 现有系统扩展
- `sys_user` 表新增 `role_type`, `managed_area`, `police_station` 字段

## 4. API 路由

| 模块 | 路径 | 方法 | 描述 |
|------|------|------|------|
| 房源管理 | /rental/houses | GET/POST | 获取/创建出租屋 |
| 房源管理 | /rental/houses/{id} | GET/PUT/DELETE | 出租屋详情/更新/删除 |
| 房源管理 | /rental/houses/nearby | GET | 附近出租屋查询 |
| 房源管理 | /rental/houses/export | GET | 导出数据 |
| 住户管理 | /rental/residents | GET/POST | 获取/创建住户 |
| 住户管理 | /rental/residents/{id} | GET/PUT/DELETE | 住户详情/更新/删除 |
| 住户管理 | /rental/residents/{id}/face | POST | 上传人脸照片 |
| 通行记录 | /rental/access-logs | GET | 获取通行记录列表 |
| 通行记录 | /rental/access-logs/heatmap | GET | 通行热力图数据 |
| 告警管理 | /rental/alerts | GET | 获取告警列表 |
| 告警管理 | /rental/alerts/{id} | GET/PUT | 告警详情/处理 |
| 告警管理 | /rental/alerts/{id}/verify | POST | 提交核查结果 |
| 统计分析 | /rental/statistics/dashboard | GET | 仪表盘统计数据 |
| 设备管理 | /rental/devices | GET/POST | 获取/创建设备 |
| 设备管理 | /rental/devices/{id} | GET/PUT | 设备详情/更新 |
| 数据推送 | /rental/push/access-log | POST | 设备推送通行记录 |
| 地理编码 | /rental/geocoder | POST | 地址转坐标 |
