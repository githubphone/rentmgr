# 出租屋管理系统开发计划

> 更新时间：2026-04-07

## 一、项目概述

出租屋管理系统是基于 RuoYi-Vue-Plus 框架二次开发的出租屋管理平台，主要功能包括房源管理、住户管理、通行记录、告警管理、设备管理等模块。

## 三、已完成任务

### 3.1 统计分析模块 (RentalStatistics) ✅

**已完成文件**：
- `RentalStatisticsController.java`
- `IRentalStatisticsService.java`
- `RentalStatisticsServiceImpl.java`
- `RentalStatisticsMapper.java`
- `RentalStatisticsMapper.xml`

**API 接口**：
- `GET /rental/statistics/dashboard` - 仪表盘统计
- `GET /rental/statistics/today` - 今日统计
- `GET /rental/statistics/device` - 设备统计
- `GET /rental/statistics/alert/month` - 本月告警统计
- `GET /rental/statistics/stranger/top` - 陌生人TOP10
- `GET /rental/statistics/house/byTownStreet` - 按镇街统计
- `GET /rental/statistics/house/byLevel` - 按等级统计
- `GET /rental/statistics/access/trend` - 通行趋势
- `GET /rental/statistics/alert/trend` - 告警趋势
