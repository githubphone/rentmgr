# 出租屋管理系统数据库设计说明

## 1. 概述

本文档描述了出租屋管理系统的数据库设计，包括表结构、字段说明、索引、外键关系以及字典表信息。

- **数据库类型**: PostgreSQL
- **创建日期**: 2026-03-26
- **版本**: 1.1.0

## 2. 表结构依赖顺序

按以下顺序执行建表脚本：

1. rental_grids (无依赖)
2. rental_houses -> rental_grids
3. rental_devices (无依赖)
4. rental_managers (无依赖，但关联sys_user)
5. rental_house_inspections -> rental_houses
6. rental_house_devices -> rental_houses, rental_devices
7. rental_residents -> rental_houses
8. rental_face_photos -> rental_residents
9. rental_access_logs -> rental_houses
10. rental_alerts -> rental_access_logs, rental_houses
11. rental_verifications -> rental_alerts
12. rental_push_data (无依赖)
