## 5. 表关系依赖顺序

1. **rental_houses** (无依赖)
2. **rental_rooms** -> rental_houses
3. **rental_residents** -> rental_houses, rental_rooms
4. **rental_face_photos** -> rental_residents
5. **rental_access_logs** -> rental_houses
6. **rental_alerts** -> rental_access_logs, rental_houses
7. **rental_verifications** -> rental_alerts
8. **rental_devices** -> rental_houses
