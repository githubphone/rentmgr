# 水口出租屋人脸识别管理模块技术设计文档

## 1. 概述

### 1.1 项目背景

为提升水口地区出租屋管理的社会治理能力，本模块基于人脸识别技术实现对陌生人的自动发现和告警。通过在出租屋入口部署人脸采集设备，系统能够实时识别进出人员，并与已登记的住户人脸库进行比对。一旦发现未登记人员，系统自动记录并推送告警至相关人员进行核查处置。

### 1.2 项目目标

- 建立完整的出租屋和住户人脸数据库，实现底数清、情况明
- 通过人脸识别技术自动化发现陌生人，提高治理效率
- 建立高效的告警推送和核查处置流程，形成闭环管理
- 为公安部门提供有价值的人员活动线索

### 1.3 涉众分析

| 涉众 | 职责 | 需求 |
|------|------|------|
| 街道办管理人员 | 系统整体运营、决策、数据分析 | 宏观统计、异常告警、报表输出 |
| 网格员 | 日常巡查、告警核查、现场处置 | 告警接收、任务派发、核查上报 |
| 房东/物业 | 房源管理、住户登记、配合核查 | 房源录入、住户管理、通知接收 |
| 派出所工作人员 | 异常人员核查、案件协查 | 人员核查、轨迹查询、数据共享 |

**功能列表：**
- 告警列表（待处理/处理中/已处理）
- 告警详情查看
- 告警处理（转交/标记处理）
- 核查结果录入
- 告警统计（按时间/区域/处理状态）

**告警处理流程：**
```
发现陌生人 → 生成告警 → 推送至网格员
    ↓
网格员接单 → 电话核实/上门核查
    ↓
选择结果：真正的陌生人/住户误识别/快递员/访客
    ↓
录入核查结果 → 完成处理
    ↓
必要时通知房东/派出所
```

### 5.7 房东端小程序模块

**设计理念：极简操作，减负提效**

房东通过微信小程序使用系统，扫码绑定设备后，系统自动完成大部分工作。

**功能列表：**

1. **扫码绑定出租屋** - 微信扫描设备二维码，自动识别并绑定

2. **住户自主登记** - 住户自助录入信息，房东只需审核确认

3. **AI辅助信息采集** - 拍照身份证自动识别、人脸自动比对公安库

4. **智能提醒推送** - 租客到期前提醒、设备离线通知

5. **通行动态感知** - 微信接收家人通行通知、陌生人告警（可选）

6. **一键联系网格员** - 快速上报，配合社区治理

**房东端页面结构：**

services:

    ports:
      - "80:80"
    depends_on:
